# Lab 001 - Linux Basics

## Objective

Learn basic Linux navigation and file management commands.

## Commands Practiced

* pwd
* ls
* cd
* mkdir
* touch
* cp
* mv
* rm

## Steps Performed

1. Checked the current working directory using `pwd`.
2. Listed files and directories using `ls`.
3. Navigated between directories using `cd`.
4. Created a new directory using `mkdir`.
5. Created a file using `touch`.
6. Copied files using `cp`.
7. Renamed and moved files using `mv`.
8. Deleted files using `rm`.
