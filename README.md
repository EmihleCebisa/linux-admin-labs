# Linux Administration Labs

A collection of hands-on Linux administration labs documenting practical system administration and cybersecurity skills.

## About

This repository contains structured Linux labs completed in a virtualized environment to build foundational knowledge in Linux administration, security, and troubleshooting.

The goal is to develop practical skills relevant to:

* Linux Administration
* Cybersecurity
* Security Operations (SOC)
* Digital Forensics
* System Troubleshooting

## Environment

* Host OS: Windows 11
* Virtualization Platform: VirtualBox
* Guest OS: Ubuntu Linux
